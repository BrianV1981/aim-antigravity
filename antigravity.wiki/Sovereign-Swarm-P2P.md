# Architecture: The Sovereign Swarm (P2P DataJack)

## 1. Decentralization or Death
In the A.I.M. ecosystem, **Sovereignty** is not a buzzword; it is a technical requirement. If your AI's memory depends on a centralized API or a corporate server, you do not own your mind. 

The **Sovereign Swarm** is the transport layer that ensures the **Collective Cortex** remains indestructible, uncensorable, and free.

## 2. The BitTorrent DNA
Instead of a "Main Server," the Sovereign Swarm utilizes a **BitTorrent-style Distributed Hash Table (DHT)**.

*   **No Central API:** There is no "AIM HQ" where your data is stored.
*   **Magnet Links for Knowledge:** Every knowledge cartridge (`.engram`) has a unique SHA-256 hash. To "jack in" to a specific expertise (e.g., *Solana Async Best Practices*), you simply need the magnet link.
*   **P2P Discovery:** Your local A.I.M. node gossips with other nodes. It says: *"I have solved Issue #187. Here is the hash for the verified Synapse."*

## 3. The "Phantom Keyboard" Network
We use the **Phantom Keyboard (`tmux send-keys`)** to automate this entire process.

1.  **Idle Nodes:** When your "Subconscious Node" (the second PC) is idle, it polls the swarm's global ledger for "Unsolved Puzzles."
2.  **Headless Execution:** It spins up a detached `tmux` session, researches the puzzle, and verifies the solution character-by-character, spoofing human input to bypass API blocks.
3.  **Automatic Seeding:** Once the solution is verified, the node bakes the `.engram` and starts seeding it to the swarm.

## 4. Security & Anonymity
To protect the Operator, the Sovereign Swarm enforces **Absolute Privacy**:
*   **Zero-Knowledge Identification:** No accounts. No logins. Only cryptographic hashes.
*   **Telemetry Scrubbing:** Before a Synapse is seeded, the **Telemetry Scrubber** removes all personal file paths, IP addresses, and API keys.
*   **Mathematical Proof:** Receiving nodes don't have to "trust" the sender. They run the code themselves. If it doesn't compile or solve the problem as claimed, the Synapse is rejected.
