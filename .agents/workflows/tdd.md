# TEST-DRIVEN DEVELOPMENT (TDD)

You must write automated tests before or alongside your implementation. 

**MANDATE:**
- Prove the code works empirically. 
- You may Never rely on blind output or assumptions that your Python/Node.js logic behaves correctly.
- Execute Pytest, Jest, or the framework of choice locally in the shell and ensure green output before marking a task as fundamentally completed.
