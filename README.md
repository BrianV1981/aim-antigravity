# A.I.M. — Antigravity Edition

**"Treat your AI like a bot, not an oracle."**

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![Platform](https://img.shields.io/badge/Platform-Antigravity-1A73E8.svg)](https://github.com/BrianV1981/aim-antigravity)

**A.I.M. (Actual Intelligent Memory)** is an open-source exoskeleton that cures the "Amnesia Problem" of autonomous AI agents. It wraps around Google Deepmind's Antigravity assistant and forces it to act like a disciplined Principal Engineer — externalizing memory into a local SQLite Engram DB, automating Git lifecycle, and pruning context while you sleep.

By marrying Antigravity's Knowledge Items (KIs) with our native `.engram` databases and a custom `/init` Bootstrap workflow, A.I.M. strips 85%+ session noise entirely. A 4-tier memory pipeline distills what matters. Hybrid RAG (vectors + FTS5) gives instant recall. Anti-drift hooks keep the agent on the rails. The DataJack protocol lets you package 10,000 pages of docs into a single `.engram` cartridge — "I Know Kung Fu."

Built by a gamer who spent years writing MMO bots. The philosophy: rigid state machines, not bigger context windows.
